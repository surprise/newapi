**NAME STATUS:**

Method:GET 
URL: https://api.minecraftservices.com/minecraft/profile/name/asd/available

You only have to send the bearer token aswell as the typical headers

if taken/blocked:
```
{
  "status" : "DUPLICATE"
}
```

if available:
200 Ok
```
{
  "status" : "AVAILABLE"
}
```

if innappropriate:
```
{
  "status" : "NOT_ALLOWED"
}
```
